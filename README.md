# LFG Task Board

A clean, minimal Kanban board built with your brand colors.

## How to Use

1. **Open it:** Just open `index.html` in your browser
2. **Add tasks:** Click "+ Add Task" or the "New Task" button (or press `Cmd/Ctrl + N`)
3. **Move tasks:** Drag and drop between columns
4. **Edit/Delete:** Hover over a card to see the action buttons
5. **Filter:** Use the category buttons to filter by LFG, Personal, or Geeves tasks
6. **Export:** Click Export to download your tasks as JSON
